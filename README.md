# Unofficial MapLibre GL JS Rust bindings

[![Crates.io](https://img.shields.io/crates/v/maplibre-gl-js?)](https://crates.io/crates/maplibre-gl-js)
[![docs.rs](https://img.shields.io/docsrs/maplibre-gl-js)](https://docs.rs/maplibre-gl-js)
[![GitHub License](https://img.shields.io/github/license/rabadandotdev/maplibre-gl-js-rs)](https://github.com/RabadanDotDev/maplibre-gl-js-rs/blob/main/LICENSE.txt)

Bindings with an ergonomic interface to [MapLibre GL JS](https://maplibre.org/maplibre-gl-js/docs/).

This is a different project to [maplibre-native-rs](https://github.com/maplibre/maplibre-native-rs), which binds to [MapLibre Native](https://maplibre.org/projects/native/)
